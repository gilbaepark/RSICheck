# RSI Check - 매매 신호 분석 대시보드

RSI(Relative Strength Index) 지표를 활용한 주식 매수/매도 신호 추천 웹 애플리케이션입니다.

## 📊 주요 기능

- **3개의 RSI 지표 분석**: 단기(9일), 중기(14일), 장기(26일) RSI를 동시에 분석
- **5개 주요 종목 모니터링**: Tesla(TSLA), Nvidia(NVDA), KORU(069500.KS), SOXL, TQQQ
- **실시간 매매 신호**: 강력매수/매수/관망/매도/강력매도 신호 제공
- **인터랙티브 차트**: Plotly 기반의 인터랙티브 가격 및 RSI 차트
- **전체 종목 요약**: 모든 모니터링 종목의 신호를 한눈에 확인
- **다양한 조회 기간**: 1개월, 3개월, 6개월, 1년 선택 가능

## 🚀 로컬에서 실행하기

### 1. 저장소 클론

```bash
git clone https://github.com/gilbaepark/RSICheck.git
cd RSICheck
```

### 2. 가상환경 생성 및 활성화 (권장)

```bash
# Windows
python -m venv venv
venv\Scripts\activate

# macOS/Linux
python3 -m venv venv
source venv/bin/activate
```

### 3. 의존성 패키지 설치

```bash
pip install -r requirements.txt
```

### 4. 애플리케이션 실행

```bash
streamlit run app.py
```

브라우저에서 자동으로 `http://localhost:8501`이 열립니다.

## 📚 RSI 지표 해석 가이드

### RSI란?

RSI(Relative Strength Index)는 주식의 과매수/과매도 상태를 나타내는 모멘텀 지표입니다.

### RSI 수치 해석

- **0~30**: 과매도 구간 (매수 신호 가능성)
- **30~70**: 중립 구간
- **70~100**: 과매수 구간 (매도 신호 가능성)

### 3개 RSI 조합 분석

이 앱은 3개의 RSI를 조합하여 더 정확한 신호를 제공합니다:

- **단기 RSI (9일)**: 빠른 추세 변화 감지
- **중기 RSI (14일)**: 일반적인 추세 확인
- **장기 RSI (26일)**: 장기적인 추세 파악

## 🎯 매매 신호 해석 방법

### 강력 매수 신호

**조건:**
- 단기 RSI가 30 이하에서 상승 반전
- 중기 RSI가 30~50 구간에서 상승 중
- 장기 RSI가 50 이상

**의미:** 과매도 구간에서 반등 시작, 강력한 매수 타이밍

### 매수 신호

**조건:**
- 단기 RSI 30 이하
- 중기 RSI 40 이하

**의미:** 과매도 구간, 매수 고려 시점

### 관망 신호

**조건:**
- 위의 매수/매도 조건에 해당하지 않음

**의미:** 명확한 신호 없음, 대기 추천

### 매도 신호

**조건:**
- 단기 RSI 70 이상
- 중기 RSI 60 이상

**의미:** 과매수 구간, 매도 고려 시점

### 강력 매도 신호

**조건:**
- 단기 RSI가 70 이상에서 하락 반전
- 중기 RSI가 70 이상에서 하락 시작
- 장기 RSI가 70 이상

**의미:** 과매수 구간에서 하락 시작, 강력한 매도 타이밍

## ⚠️ 주의사항

### 투자 책임

- **이 앱은 투자 참고 정보만 제공하며, 투자 권유가 아닙니다.**
- 실제 투자 결정은 본인의 판단과 책임 하에 이루어져야 합니다.
- 과거 데이터는 미래 수익을 보장하지 않습니다.

### 데이터 출처

- 모든 데이터는 Yahoo Finance에서 제공받습니다.
- 데이터 정확성과 실시간성은 Yahoo Finance의 서비스 품질에 따라 달라질 수 있습니다.

### 기술적 분석의 한계

- RSI는 여러 기술적 지표 중 하나일 뿐입니다.
- 다른 지표, 뉴스, 기업 실적 등을 종합적으로 고려해야 합니다.
- 시장 상황에 따라 RSI 신호의 유효성이 달라질 수 있습니다.

### 레버리지 ETF 주의

- SOXL, TQQQ는 3배 레버리지 ETF로 변동성이 매우 큽니다.
- 단기 트레이딩 전용이며, 장기 보유 시 손실 위험이 큽니다.
